## Mandatory Links Per Page Type

### Supporting Pages (Syllabus, Pattern, Cutoff, Salary, Previous Papers)

Each must contain:

| Link Target | Placement | Example |
|------------|-----------|---------|
| Authority page (notification) | First paragraph + sidebar | `/ssc-cgl-notification` |
| Cluster hub | Breadcrumb + footer links | `/ssc-jobs` |
| Site-wide hub | Breadcrumb | `/sarkari-jobs` |
| 2 sibling supporting pages | "Related" section | `/ssc-cgl-exam-pattern`, `/ssc-cgl-salary` |
| 1 cross-cluster related page | "You may also like" | `/rrb-ntpc-syllabus` (if current is `/ssc-cgl-syllabus`) |

**Minimum links per supporting page:** 6

### Authority Pages (Notifications)

| Link Target | Placement | Example |
|------------|-----------|---------|
| All supporting pages | Table of Contents / Quick Links | `/ssc-cgl-syllabus`, `/ssc-cgl-exam-pattern`, etc. |
| Cluster hub | Breadcrumb + first paragraph | `/ssc-jobs` |
| Site-wide hub | Breadcrumb | `/sarkari-jobs` |
| 2 other authority pages in cluster | "Related Exams" section | `/ssc-chsl-notification`, `/ssc-mts-notification` |
| 1 cross-cluster authority page | "Similar Exams" | `/rrb-ntpc-notification` |

**Minimum links per authority page:** 10

### Cluster Hub Pages

| Link Target | Placement | Example |
|------------|-----------|---------|
| All authority pages in cluster | Main content listing | All SSC exam notification links |
| All other cluster hubs | "Explore More" footer grid | `/railway-jobs`, `/banking-jobs`, etc. |
| Site-wide hub | Breadcrumb | `/sarkari-jobs` |
| Top 3 supporting pages by volume | Featured section | `/ssc-cgl-syllabus`, `/ssc-cgl-salary` |

**Minimum links per hub page:** 15

## Anchor Text Guidelines

1. **Primary anchor:** Use exam name + page type (e.g., "SSC CGL Syllabus 2026")
2. **Variation anchors:** Use natural phrases (e.g., "check the complete syllabus", "view salary details")
3. **Never use** "click here" or "read more" as anchor text
4. **Max 3 links** to the same target page from a single page
5. **Anchor text distribution:** 60% exact match, 30% partial match, 10% branded/natural

## Contextual Link Placement

### In-Content Links (Body Text)

- First mention of a related exam: link to its notification page
- First mention of syllabus/pattern/salary: link to the respective supporting page
- First mention of a state: link to the state filter page
- Maximum 1 link per 100 words in body content
