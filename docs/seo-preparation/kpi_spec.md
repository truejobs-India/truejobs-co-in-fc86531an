# KPI Specification — Government Jobs SEO

## Primary KPIs

| Metric | Definition | Target (90 days) | Measurement |
|--------|-----------|-------------------|-------------|
| Indexed Pages | Pages appearing in Google index | 120 pages indexed | GSC → Coverage |
| Organic Impressions | Times pages appeared in search results | 500,000 monthly | GSC → Performance |
| Organic Clicks | Clicks from search results | 25,000 monthly | GSC → Performance |
| Average Position | Mean ranking across target keywords | <15 for Priority 1 clusters | GSC → Performance |
| Click-Through Rate (CTR) | Clicks / Impressions | >3.5% average | GSC → Performance |
| Organic Sessions | Users arriving via organic search | 20,000 monthly | GA4 |

## Secondary KPIs

| Metric | Definition | Target | Measurement |
|--------|-----------|--------|-------------|
| Page Speed (LCP) | Largest Contentful Paint | <2.5s on mobile | PageSpeed Insights |
| Core Web Vitals Pass | CWV passing for all pages | 100% | GSC → CWV report |
| Bounce Rate | Single-page sessions | <65% | GA4 |
| Pages per Session | Average pages viewed | >1.8 | GA4 |
| Time on Page | Average engagement time | >45 seconds | GA4 |
| FAQ Rich Results | Pages showing FAQ snippets | >30% of FAQ pages | GSC → Enhancements |
| Internal Click Depth | Average clicks to reach any page | <3 from homepage | Screaming Frog |

## Measurement Schedule

| Frequency | Action |
|-----------|--------|
| Daily | Check GSC for crawl errors, new indexed pages |
| Weekly | Review impressions, clicks, CTR per cluster |
| Bi-weekly | Compare position changes for target keywords |
| Monthly | Full KPI report with trend analysis |
| Quarterly | Content audit — update stale pages, review KPI targets |
